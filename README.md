# 镜像生成器 · Mirror Generator

> 不是复制一个"赛博你"去教别人,而是造一面镜子:拿着你自己的性格、退路、软肋和价值排序,在你面临真实选择时反照出来——拦冲动、逼算账、点破盲点。

## 它解决什么

大多数人做重大决定(去哪工作、花大钱、换行、人际边界)时,会被情绪和冲动带跑,又没人给兜底。这个 skill 把"照镜子"这件事**固化成一套可复用的流程**,让你每次都能被自己的价值观校准,而不是被当下的情绪推着走。

> 注意:它提供的是**方法 + 逻辑**(一份剧本),真正"照镜子"靠的是运行它的 AI 助手(如 WorkBuddy)。它不含任何个人隐私。

## 和"数字分身"的区别

| | 数字分身 / 赛博永生 | 镜像生成器 |
|---|---|---|
| 目的 | 复制一个你去对外教人 | 拿你的标尺反过来校准**你** |
| 方向 | 对外输出 | 对内照自己 |
| 隐私 | 常需公开你的信息 | 零隐私,只存你本地 |

## 安装

**方式一(推荐):解压即用**
把 `mirror-generator.zip` 解压到 WorkBuddy 的 skills 目录:

- 用户级:`~/.workbuddy/skills/`
- 或项目级:`<你的项目>/.workbuddy/skills/`

**方式二:手动复制**
直接把整个 `mirror-generator/` 文件夹复制过去即可。

安装后,在对话里说下面任意一句就能触发。

## 怎么用

**触发话术**

- "建一个我的镜像"
- "自我蒸馏"
- "用镜像看看这个选择"

**第 1 步:选入口(你自选,不强制)**

- **A 问答式** — 快、清晰,一次性问核心维度
- **B 聊天式** — 零负担,正常聊,AI 被动提取,档案自然长出来
- **C 素材导入式** — 丢日记 / 笔记 / 聊天记录导出,AI 直接读、提炼
- **D 场景选边式** — 给处境让你站队,AI 推断你的决策规则
- **E 智能推荐** — 不挑,AI 按你手头素材 / 意愿自动选最省力路径

**第 2 步:建 4 层档案**(最小可用 = ② + ④,①③ 慢慢补)

- ① **我是谁** — MBTI / 人际模式 / 决策风格 / 风险态度 / 能量来源 / 价值排序
- ② **我的底** — 退路硬底(不可归零)+ 绝对红线
- ③ **我的坑** — 决策软肋 + 情绪触发点 + 核心恐惧
- ④ **我的尺** — 价值排序 + 好选择清单 / 坏结果清单

档案存为你**本地**的一个文件(如 `自我蒸馏.md`),不公开、不上传。

**第 3 步:用镜像(泼冷水)**

面临选择时说"用镜像看看 X",会收到:

- 坏结果清单(踩了哪些会完)
- 好选择清单(符合长期利益长啥样)
- 分条、像真人聊天般逐条泼冷水,**不替你拍板**
- 自评反查:你自填的标签若和真实行为矛盾,会轻轻点出
- 底线校核:退路不可归零

**第 4 步:扩展**

档案是活文档。有空再往"扩展区"补细节(情绪模式、能量消耗、过往复盘、沟通偏好…),越用越准。

## 隐私

- 所有个人数据**只存在你本地**,不上传、不联网、不发给任何人。
- skill 模板是空的方法,不含任何人的隐私,可安全开源。
- 开源 / 分享时,请**只分享模板**,别把你的私人档案发出去。

# 性格特征标签库

轻量自评,随时可填(前/中/后都行)。每维挑最贴的,可多选或自填。这是所有建档入口的共用字段。

## MBTI / 认知风格
- 4 字母(如 INFP);或单维:思考(T)/情感(F)、直觉(N)/实感(S)、判断(J)/感知(P)、内倾(I)/外倾(E)
- 注意:自评常与行为不符,留待镜像反查点出。

## 人际模式
- 讨好型(对外也让利)/ 对内让利对外稳 / 边界感强 / 内敛 / 外放

## 决策风格
- 冲动型 / 算账型 / 回避型 / 跟风型 / 算账+小范围试探(B 型)

## 风险态度
- 保守 / 冒进 / 摇摆 / 留退路不归零

## 能量来源
- 内倾(独处充电)/ 外倾(社交充电)

## 价值排序
- 用 ">" 链列出:如 健康 > 家人 > 财富
- 底线:法律红线(任何情况不碰)

# 入口 A · 问答式

适合:想要快、要清晰、不在乎填表的人。

做法:
1. 先只问 **② 我的底** 与 **④ 我的尺**(最小可用),各 1–2 题:
   - "有什么是你绝对不能失去的?(家/钱/技能/人)"
   - "你最看重什么?排个序。"
   - "踩了什么你会觉得'完了'?"
2. 用户答完即可开工。然后按需问 ①(性格特征,对照 `personality-traits.md` 打标签)与 ③。
3. 每问完一块,把结果写进档案文件,并回显确认。
4. 不打听隐私细节;红线/退路只记"类别",不记具体金额账号。

提醒:问答式易让人烦,**控制题量**,能聊出来的别问。
